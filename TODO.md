# GHOZTWOODS LLC WEBSITE - BUILD TRACKER

## Project Overview
Building a world-class MSP website with technician tracking and scam detection tools.

**Color Scheme:**
- Primary Green: #00ff00 (Matrix/Terminal "Systems Go")
- Accent Blue: #00d4ff (Security/Protection)
- Background: #0a0a0a (Deep Black)

**Technology Stack:**
- Frontend: HTML5, Tailwind CSS, Leaflet.js
- Backend: Python (Flask), geo_tracker.py
- Deployment: GitHub Pages
